## 📖 소개

Lumina Daily는 **Google Gemini AI**가 큐레이션한 전 세계의 명언과 지혜를 매일 아름다운 이미지와 함께 전달하는 앱입니다. 단순한 명언 앱을 넘어, AI가 그 속에 담긴 지혜의 깊이를 해설하고 당신의 삶에 적용할 수 있는 성찰을 제공합니다.

## 🚀 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 테스트
npm test
```

### 환경 변수 설정

`.env.example`을 복사하여 `.env` 파일을 생성하고 Firebase 설정값을 입력하세요.

```bash
cp .env.example .env
```
